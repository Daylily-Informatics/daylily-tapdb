Getting Around

Use the main navigation to browse templates, instances, lineage links, and the graph view.

  • Home: summary counts and quick links
  • Templates: template catalog and template metadata
  • Instances: instance list and object details
  • Lineages: directed parent/child relationships
  • Graph: interactive graph visualization and edge editing

Graph Gestures

  • D + right click: delete clicked node or edge
  • 3x left click on node: child-wave glow (pink)
  • 3x right click on node: parent-wave glow (aqua)
  • L + left click on node: select child, then click parent to create edge
